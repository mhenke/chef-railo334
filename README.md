DESCRIPTION
===========

Sets up Railo 3.3.4.003. Currently supports Railo Express only.

REQUIREMENTS
============

ATTRIBUTES
==========

For Railo
--------------

* `node['railo334']['install_path']` - Railo installation path (default: "/opt/railo")
* `node['railo334']['webroot']` - Railo webroot (default: "/vagrant/wwwroot")
